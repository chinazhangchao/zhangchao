intoxicate /in'tɒksikeit/ [in-;toxic;-ate v.] v.使醉； 使陶醉； 使中毒

contumacy /'kɒntjumәsi/ [con-(com-);tum;-acy n.] n.抗拒，顽抗； 蓄意藐视法庭

单词 音标 词根 释义
